## Run Locally

Clone the project

```bash
  git clone https://github.com/Vedarth1/RBAC
```

Go to the project directory

```bash
  cd RBAC
```

### Run Backend

```bash
  npm install
```

```bash
  npm run dev
```

### Run Frontend

Go to the Client directory

```bash
  cd Client
```
Install dependencies

```bash
  npm install
```

start your frontend service

```bash
  npm run dev
```
